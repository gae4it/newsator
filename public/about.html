About Newsator AI

Project Overview

Newsator AI is a student project developed for educational and research purposes. It demonstrates the integration of modern AI technologies for real-time news aggregation and summarization.

Purpose

This project was created to explore and demonstrate:

  • AI-powered content aggregation using Google Gemini API
  • Real-time news summarization with Google Search grounding
  • Modern web application architecture and design patterns
  • Responsive UI/UX design principles
  • Integration of third-party AI services

Technology Stack

  • Frontend: React, TypeScript, Tailwind CSS
  • AI Service: Google Gemini 2.5 Flash with Search Grounding
  • Build Tool: Vite
  • Hosting: Netlify

Features

  • 12 geographic regions (World, Europe, USA, Asia, and more)
  • 10 news categories (Breaking News, Politics, Economy, Technology, etc.)
  • Real-time AI-powered news summarization
  • Instagram Stories-style circular region selector
  • Responsive design optimized for mobile and desktop
  • Dark mode support

Privacy & Data

This application:

  • Does not collect or store any user data
  • Does not use cookies or tracking
  • Does not require user registration
  • Fetches news in real-time without caching

Disclaimer

This is an educational project and is not intended for commercial use. News content is generated through Google Gemini API and accuracy is not guaranteed. This application is provided "as is" for demonstration purposes only.
